Sparanoid is a personal website created by Tunghsiao Liu (a.k.a. 劉通曉), an undergraduate at Tianjin Academy of Fine Arts. You are welcome to ping me on Twitter or send correspondence to info@sparanoid.com.

Projects

  • 7-Zip L10n Project - An open-source file archiver. I maintain Chinese Simplified localization since 2004.
  • Quotmarks Replacer - A WordPress plugin disables wptexturize function that keeps all quotation marks and suspension points in half-width form.

More small projects please visit Work page.
